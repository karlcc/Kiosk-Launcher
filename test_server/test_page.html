üìù Test PHP Script

To test device validation, create a PHP file on your server with the provided code. The script will:

  • Receive device headers from your kiosk app
  • Check against a whitelist of device IDs
  • Log all access attempts
  • Return JSON response indicating access status

üîß Setup Instructions

  1. Upload device_whitelist.php to your web server
  2. Add your device ID to the whitelist array in the PHP file
  3. Set your kiosk app to load a URL that makes requests to the PHP script
  4. Enable device validation in the kiosk app (long-press Save URL button)
Note: When device validation is enabled, all HTTP requests from the WebView will include device identification headers. Your PHP script can then validate these headers against your whitelist.
